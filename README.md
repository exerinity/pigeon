# Run it yourself
The source code here is not intended to be self-hosted; it is provided here for transparency and basically showing people how it works. If you want to run it yourself, go ahead, but no help will be provided. Nevertheless, it isn't (that) difficult to set up. **The MIT license explicitly strips all liability from me, you are on your own.**

# Dependencies
- Node.js
- Discord.js
- Google AI for Gemini
- Your eyes and brain
- Amazon S3 client for [the history command](/commands/history.js) (it actually uses Cloudflare R2)
- And obviously, dotenv, the backbone of fucking everything
(see [package.json](package.json))
